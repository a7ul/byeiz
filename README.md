# To run locally:

1. `git clone` this repo
2. `npm install`
3. `npm start`

This should launch the server in port:3000

After that open another terminal and do:

**Simple Example**

`curl http://localhost:3000/`

or

`wget -q -O - http://localhost:3000/`
